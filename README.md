# Images

A simple repository to stash and organize images.

## Structure

This repository is organized into the following directories:

- `screenshots/` - Screenshots and screen captures
- `icons/` - Icons and small graphics
- `photos/` - Photos and larger images
- `logos/` - Logos and branding materials
- `misc/` - Miscellaneous images that don't fit other categories

## Usage

1. Upload images to the appropriate directory based on their type
2. Use descriptive filenames for easy identification
3. Images are organized by category for easy browsing
